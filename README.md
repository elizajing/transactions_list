Introduction
============
This is an application showing transactions to accounts, you'll also be able to add a transaction to a specific account.

How to Run
----------
git clone this repo
<p><code>npm i</code> </p>
<p><code>cd app-frontend</code></p>
<p><code>npm i</code></p>

Node version: v18.15.0.

Run e2e Tests
-------------
`npm run test`
